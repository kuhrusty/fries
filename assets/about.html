This is a support app for Cheapass Games' Lord of the Fries. Its purpose is to display the ingredients for the current order large enough for everyone at the table to see.

Almost all of the text & images are property of Cheapass Games and are, uhh, sort of used without permission. So, uhh, that's not exactly cool.

There are a couple ways to use this thing. One is to manually enter the ingredients when an order is rolled or called. Another is to use the Call button to call an order, or the Roll button (which looks like a fast food icon... I haven't made a d6 icon yet) to roll an order.

The -1 button is just for keeping track of how many times an order has gone around the table.

This program is open source! If you want to fiddle with the source code, it's on GitHub.
